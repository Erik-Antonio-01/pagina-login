# 🔐 Sistema de Login

Desenvolvi este projeto como prática de desenvolvimento front-end, criando uma interface de login simples utilizando HTML e CSS.

## 📄 Funcionalidades

* Tela de login *(interface apenas, sem validação real)*
* Página de recuperação de senha *(simulação)*
* Página de criação de conta *(simulação)*
* Página pós-login *(simulação)*
* Navegação entre páginas

> ⚠️ Este projeto não possui integração com backend, sendo apenas uma simulação visual.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts

## 📁 Estrutura do projeto

```
pagina-login/
│
├── index.html
├── index_esqueceu.html
├── index_login.html
├── index_criar.html
├── style.css
```
